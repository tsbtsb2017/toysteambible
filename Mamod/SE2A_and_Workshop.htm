Owner's website:
Notes: This engine is complete and is a fully working model. It has the more modern type whistle, a pressure relief valve, and side boiler filler. Note there are 3 brass boiler pipes, one input to the piston head and two relief to the chimney stack, which are required for this foward / reverse engine. On this model the meths burner is set though a matching hole in the base plate, and is facing outwards away from the balance wheel. The engine is on a display board running a full set of Mamod tools circa 1960s 1970s and they are :- Drop Hammer, Vertical press, double Grinding Wheels, and double Polishing wheels, all driven via a Mamod line shaft.
